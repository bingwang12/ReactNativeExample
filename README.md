# React Narive 简易教程
### 1.下载

可以使用 `git` 

```
git clone https://github.com/qiangxinyu/ReactNativeExample

```

或者直接下载 `zip` 然后解压


### 2.安装依赖

进入项目目录

```
cd ReactNativeExample
```

然后进入对应目录，比如 `ReactNative1`

```
cd ReactNative1
```

然后安装依赖

```
npm install
```

然后安装用到的第三方

```
npm install react-native-tab-navigator --save
```

### 3.运行

```
react-native run-ios
```
